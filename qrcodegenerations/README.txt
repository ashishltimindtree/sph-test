# How to use qrcodegenerations module.

- Git clone module under "web/modules/custom"
- Enable module using command
refer command "drush en qrcodegenerations"
- Run command "composer require chillerlan/php-qrcode" to download the php library

## What does the module do.
    - Cretaes product content type.
    - Creates block and place it to side bar region.
